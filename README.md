# readme-dynamic

Welcome to the readme-dynamic backend plugin!

_This plugin was created through the Backstage CLI_

## Getting started

``` 
    yarn clean
    yarn tsc
    yarn build
    yarn export-dynamic
    cd dist-dynamic
    npm pack --json > ../npminfo.json
    npm publish anattama-backstage-plugin-readme-dynamic-backend-dynamic-0.1.3.tgz
```
